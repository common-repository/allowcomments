Allow comments to posts with custom field "allow_comments" even if option 'close comments to old posts' is on.

== Description ==

Allow comments to chosen posts even if option "_close comments to old posts_" is on.

Разрешает комментирование отдельных записей при включенной опции закрытии комментариев после определенного срока (ВП 2.7). Для этого надо включить чекбокс "Allow comments to this post".

Работает как с плагином для версий ниже 2.6, так и с функцией по умолчанию в 2.7.

== Installation ==

1. Upload `AllowComments` to the `/wp-content/plugins/` directory
2. Active plugin in menu `Plugins`.
